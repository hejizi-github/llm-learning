## 2. 关键问题：「内层的功劳/过错怎么分？」

1983 年前后，一批研究者重新拿起这个问题。他们知道答案很简单：

**加一层！**

两层神经网络能解决 XOR：
```
输入层     隐藏层     输出层
  A  ──┐         ┌── 中间节点1 ──┐
       ├── 计算 ──┤              ├── 最终答案
  B  ──┘         └── 中间节点2 ──┘
```

但「加一层」带来了一个新问题：

**如果最终答案答错了，谁的责任？**

- 输出层的权重？
- 隐藏层的权重？
- 隐藏层的权重连输出都没有，怎么知道自己做得对不对？

这叫**「功劳归因问题」（credit assignment problem）**。

这个问题从 1960 年代就困扰着研究者。1986 年的论文给出了清晰的解法。

---

## 3. 直觉：多米诺骨牌效应

想象一排多米诺骨牌：

```
骨牌1 → 骨牌2 → 骨牌3 → 最后倒下
```

最后那块骨牌倒下是谁的责任？骨牌3。  
骨牌3 倒是谁的责任？骨牌2。  
骨牌2 倒是谁的责任？骨牌1。

**「责任」可以一层一层往回追。**

这就是反向传播（Backpropagation）的核心思想：  
- 先从前往后算（前向传播）
- 算出答案之后，把「算错了多少」这个信号，从后往前一层层传回去
- 每层都根据收到的「责任信号」更新自己的权重

---

## 4. 数学：链式法则（Chain Rule）

反向传播用的数学工具只有一个：**链式法则**（初中物理里「复合运动」的数学版本）。

### 什么是链式法则？

假设：
- f(x) 表示某个函数
- g(x) 是另一个函数
- h(x) = f(g(x)) — 先做 g，再做 f

如果 x 变动了一点点，h 变化了多少？

$$\frac{dh}{dx} = \frac{df}{dg} \times \frac{dg}{dx}$$

中文读法：「h 对 x 的变化率 = f 对 g 的变化率 × g 对 x 的变化率」

### 神经网络里的链式法则

```
权重 w → 加权求和 z → 激活值 a → 损失 L
```

损失 L 对权重 w 的灵敏度：

$$\frac{\partial L}{\partial w} = \frac{\partial L}{\partial a} \times \frac{\partial a}{\partial z} \times \frac{\partial z}{\partial w}$$

每一项都能单独算，然后乘起来。这就是为什么反向传播可以一层一层往回传——每一步只需要「下一层传来的信号 × 本层自己的导数」。

> **不需要微积分也能理解**：链式法则说的是「两个放大倍数相乘」。
> 如果 g 把 x 放大了 3 倍，f 又把 g 放大了 2 倍，那 f 总共把 x 放大了 6 倍。

## 5. Sigmoid 激活函数

隐藏层不能用感知机那样的阶跃函数（0 或 1），因为阶跃函数的导数处处为 0 或无穷大，没法用链式法则。

1986 年的论文用了 **sigmoid** 函数：

$$\sigma(z) = \frac{1}{1 + e^{-z}}$$

它把任意输入压缩到 (0, 1) 之间，像一个「软版」的 0-1 开关。

更重要的是，它的导数非常简洁：

$$\sigma'(z) = \sigma(z) \times (1 - \sigma(z))$$

这让链式法则的计算变得很简单（见 notebook Cell 5）。

## 6. 为什么是 1986？

Paul Werbos 在 1974 年的博士论文里就推导过反向传播，但没有引起注意。

1986 年，Rumelhart、Hinton、Williams 在《Nature》上发表了清晰的推导和实验结果：
- 用两层网络解决了感知机无法解决的问题
- 展示了网络能「学到」有意义的内部表示
- 论文写得足够清楚，让其他研究者能实现

《Nature》是自然科学领域最顶级的期刊。这篇论文在这里发表，意味着：这不只是计算机科学的玩具，这是一个真正重要的发现。

**13 年的寒冬，就此结束。**

## 7. 用 20 行代码解决 XOR

节点 02 里，感知机在 XOR 上永远学不会。

用两层网络 + 反向传播：

```
第  100 轮 | 损失: 0.2431
第  500 轮 | 损失: 0.0312
第 1000 轮 | 损失: 0.0089
第 2000 轮 | 损失: 0.0021

XOR 准确率：100%（4/4 全对）
```

感知机学了 500 轮都不收敛，两层网络学了 2000 轮，4 个全对。

差别就在于：有了反向传播，隐藏层终于知道自己该怎么改了。
