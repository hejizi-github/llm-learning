## 故事：AI 寒冬之后的一声炮响

1969 年，Minsky 和 Papert 的书证明感知机无法学习 XOR。整个神经网络领域陷入低谷，资金断了，研究者转行了。

但问题本身没消失——它清清楚楚地摆在那里：

> **感知机不行，因为它只有一层。那两层、三层呢？有没有办法训练多层网络？**

1974 年，保罗·韦博斯（Paul Werbos）在哈佛的博士论文里已经推导出了答案，但学界几乎没人知道——他的工作被主流期刊拒绝，被当作边角料遗忘了。

1982 年前后，大卫·鲁梅尔哈特（David Rumelhart）独立重新推导出同样的算法。

1986 年 10 月，他和杰弗里·辛顿（Geoffrey Hinton）、罗纳德·威廉斯（Ronald Williams）把这个算法发表在《自然》杂志上——这是世界上最权威的科学期刊之一。

那一枪，打醒了沉睡十七年的领域。

## 直觉：错误怎么"往回流"？

想象你在练习投篮。

1. 你投了一个球（**前向传播**：输入→网络→输出）
2. 球偏左了 30 厘米（**计算误差**）
3. 你从结果往回想：是手腕偏了？还是脚步没站好？还是出手时机差了？（**反向传播**：误差→反向流过每一层→找出每个权重的"责任"）
4. 你稍微调整手腕（**更新权重**：梯度下降）
5. 重复，直到球越来越准

反向传播做的，就是步骤 3：**把最终误差，按照每个参数的"贡献度"分配回去**。

## 数学：链式法则（高中代数就够）

### 先回忆：什么是函数？

如果 `y = 2x + 1`，那么 x 变化时，y 跟着变化。  
`dy/dx = 2` 意思是"x 每增加 1，y 增加 2"。  
这叫**导数**——描述"变化率"。

> 注意：这里只要你理解"斜率"的概念就行了，不需要学微积分。我们会用最简单的形式。

### 链式法则（函数套函数）

如果 `z` 依赖 `y`，`y` 依赖 `x`——那么 `z` 怎么随 `x` 变化？

用乘法：

$$\frac{dz}{dx} = \frac{dz}{dy} \times \frac{dy}{dx}$$

**直觉**：z 对 x 的敏感度 = z 对 y 的敏感度 × y 对 x 的敏感度。

### 在神经网络里

每一层的输出就是下一层的输入。损失（Loss）是最终的误差。

```
Loss → 输出层 → 隐藏层 → 输入层
```

用链式法则，可以计算 Loss 对**每个权重**的导数（变化率）。  
知道了变化率，就知道"往哪个方向调权重能让 Loss 减小"。

这个方向就叫**梯度**，调整权重的过程叫**梯度下降**。

## 网络结构（最简单的版本）

下面的图展示了一个两层网络（输入层 + 隐藏层 + 输出层）：

```
输入 x₁, x₂
     ↓      ↓
  [隐藏层：2个节点，用 sigmoid 激活]
        ↓
  [输出层：1个节点，sigmoid]
        ↓
      预测值 ŷ
        ↓
   损失 L = mean((y - ŷ)²)
```

**为什么要 sigmoid 激活？**  
纯线性层叠加还是线性。sigmoid 引入了弯曲（非线性），让网络能学到更复杂的模式。

**sigmoid 长什么样**：

```
sigmoid(x) = 1 / (1 + e^(-x))
```

当 x 很大时，输出趋近 1；当 x 很小时，输出趋近 0；当 x = 0 时，输出恰好 = 0.5。

## 反向传播的步骤（具体）

**前向传播**（左→右计算预测）：
1. 算隐藏层输入：`h_in = W₁·x + b₁`
2. 算隐藏层输出：`h = sigmoid(h_in)`
3. 算输出层输入：`o_in = W₂·h + b₂`
4. 算预测值：`ŷ = sigmoid(o_in)`
5. 算损失：`L = mean((y - ŷ)²)`（对所有样本取平均）

**反向传播**（右→左计算梯度）：
1. dL/dŷ = -(y - ŷ)
   > 数学推导：`L = mean((y-ŷ)²) = (1/N)·∑(y-ŷ)²`，
   > 对 ŷ 求导得 `dL/dŷ = -(2/N)·(y-ŷ)`。
   > 代码里写成 `-(y-ŷ)` 是省略了常数因子 `2/N`——
   > 它只是一个正数缩放，不改变梯度方向，效果等价于把学习率缩小 `2/N` 倍。
2. dŷ/d(o_in) = ŷ·(1 - ŷ)  ← sigmoid 的导数
3. 用链式法则，算出 dL/dW₂
4. 再往前一层，算出 dL/dW₁

**更新权重**：
```
W = W - 学习率 × 梯度
```

## 解开 XOR 的谜题

节点 02 留下的悬念：XOR 对单层感知机来说是线性不可分的——**无解**。

加一个隐藏层，用反向传播训练，就能解了。

为什么？隐藏层的作用是**重新表示**输入。它把 (0,0)(0,1)(1,0)(1,1) 这四个点，映射到一个新的空间里——在那个新空间里，XOR 变成线性可分的了。

## 承上启下

反向传播让我们能训练多层网络。但问题随之而来：

- 层数一多，梯度在反向传播中会越来越小，导致早期层几乎学不到东西（**梯度消失问题**）
- 训练大网络需要大量数据和算力

这两个问题在 1990 年代让神经网络再次陷入低谷。直到 2006 年，Hinton 的"深度信念网络"和 2012 年 GPU 算力的爆发，才彻底解决。
