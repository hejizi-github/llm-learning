## 时代背景：图像生成的两条弯路

2020 年，语言模型世界里 GPT-3 刚刚证明"大力出奇迹"，但图像生成领域却仍被两个老问题困扰：

**GAN（对抗生成网络，2014）** 的麻烦：
- 训练极不稳定——生成器和判别器像两个互相抢玩具的孩子，容易陷入"你强我躲、你躲我强"的死循环
- "模式崩塌"：网络学会只生成几种看起来真实的图片，忽略其他可能性

**VAE（变分自编码器，2013）** 的麻烦：
- 生成图片模糊——因为它在像素空间做了过多的平均
- 理论优美，但输出质量始终差 GAN 一截

2020 年，Jonathan Ho 等人（UC Berkeley）发布了 DDPM 论文，用一个出人意料的思路绕开了这两个困境：**与其教网络"直接画出图片"，不如教它"把噪声一点一点擦掉"**。

> 📄 原论文：Ho, J., Jain, A., & Abbeel, P. (2020). *Denoising Diffusion Probabilistic Models*. NeurIPS 2020. arXiv:2006.11239

---

## 核心直觉：向前加噪，向后去噪

想象你有一张清晰的照片：

```
第0步：清晰照片 🖼️
第1步：轻微模糊（加了一点点噪声）
第2步：更模糊了...
...
第1000步：纯粹的白噪声 ❄️（完全看不出原图）
```

这叫**正向过程**：把图片逐渐"弄脏"，直到变成无意义的随机噪声。

神奇的是，如果你训练一个神经网络，每次只做**"稍微擦掉一点噪声"**这件小事——经过 1000 步反向操作，你就能从一张纯噪声重建出一张清晰图片。

```
纯随机噪声 ❄️
→ 稍微清楚了一点
→ 再清楚一些
...
→ 清晰图片 🖼️
```

这叫**反向过程（采样）**。

DDPM 的天才之处：把一个很难的问题（直接生成逼真图片）拆分成 1000 个很简单的子问题（每步只去掉一点点噪声）。

## 正向过程：用数学描述"逐步加噪"

### 单步加噪

设 $x_0$ 是原始图片，$x_t$ 是加了 $t$ 步噪声后的图片。每步加噪规则是：

$$
q(x_t \mid x_{t-1}) = \mathcal{N}\!\left(x_t;\; \sqrt{1 - \beta_t}\, x_{t-1},\; \beta_t \mathbf{I}\right)
$$

**数学小补丁**：$\mathcal{N}(\mu, \sigma^2)$ 表示均值为 $\mu$、方差为 $\sigma^2$ 的正态分布。这个公式说：$x_t$ 等于把 $x_{t-1}$ 稍微缩小（乘以 $\sqrt{1-\beta_t}$），再加上一点随机噪声。

其中 $\beta_t \in (0, 1)$ 叫**噪声调度**，控制每步加多少噪声。$t$ 从 1 到 $T$（论文里 $T=1000$）。

**直觉**：$\beta_t$ 很小（比如 0.0001）→ 每步只加极少量噪声 → 正向过程缓慢磨损图片。

### 关键公式：任意时间步直接采样

逐步计算 1000 步太慢了。DDPM 推导出一个关键公式，可以**从 $x_0$ 直接跳到任意步骤 $x_t$**：

$$
q(x_t \mid x_0) = \mathcal{N}\!\left(x_t;\; \sqrt{\bar{\alpha}_t}\, x_0,\; (1 - \bar{\alpha}_t)\mathbf{I}\right)
$$

等价地写成采样形式：

$$
x_t = \sqrt{\bar{\alpha}_t}\, x_0 + \sqrt{1 - \bar{\alpha}_t}\, \varepsilon, \quad \varepsilon \sim \mathcal{N}(0, \mathbf{I})
$$

其中：
- $\alpha_t = 1 - \beta_t$
- $\bar{\alpha}_t = \prod_{s=1}^{t} \alpha_s$（从第 1 步到第 $t$ 步所有 $\alpha$ 的连乘）

**手算验证**（2步简化版）：

设 $\beta_1 = 0.1$，$\beta_2 = 0.2$，$x_0 = 1.0$：

- $\alpha_1 = 0.9$，$\alpha_2 = 0.8$
- $\bar{\alpha}_2 = 0.9 \times 0.8 = 0.72$
- $x_2 = \sqrt{0.72} \times 1.0 + \sqrt{0.28} \times \varepsilon \approx 0.849 + 0.529\varepsilon$

当 $t \to T$（比如 $t=1000$）时，$\bar{\alpha}_T \approx 0$，公式变成 $x_T \approx \varepsilon$——纯噪声。完美！

## 反向过程：神经网络学习"去噪"

### 网络要预测什么？

理论上，反向过程需要计算 $p_\theta(x_{t-1} \mid x_t)$——给定加了噪声的图 $x_t$，还原出稍干净一点的 $x_{t-1}$。

DDPM 的简化发现（论文第 3.2 节）：**不需要预测 $x_{t-1}$，只需要预测被加进去的噪声 $\varepsilon$！**

直觉：既然 $x_t = \sqrt{\bar{\alpha}_t}\, x_0 + \sqrt{1-\bar{\alpha}_t}\, \varepsilon$，如果我们知道 $\varepsilon$，就能反推出 $x_0$，进而算出 $x_{t-1}$。

神经网络 $\varepsilon_\theta(x_t, t)$ 接收：
- $x_t$：当前有噪声的图片
- $t$：时间步（告诉网络现在噪声有多深）

输出：对噪声 $\varepsilon$ 的预测。

### 训练目标（简化 ELBO）

$$
\mathcal{L} = \mathbb{E}_{t, x_0, \varepsilon}\left[\left\lVert \varepsilon - \varepsilon_\theta\!\left(\sqrt{\bar{\alpha}_t}\, x_0 + \sqrt{1-\bar{\alpha}_t}\, \varepsilon,\; t\right) \right\rVert^2\right]
$$

**翻译成人话**：随机选一张图片 $x_0$，随机选一个时间步 $t$，随机采样一个噪声 $\varepsilon$，算出 $x_t$，让网络猜 $\varepsilon$，最小化猜测误差。

这就是个**均方误差（MSE）**！没有 GAN 的博弈不稳定，没有 VAE 的模糊上界——只是预测噪声。

### 训练算法

```
重复：
  1. 从数据集取一张图 x_0
  2. 随机选时间步 t ∈ {1, ..., T}
  3. 采样噪声 ε ~ N(0, I)
  4. 计算带噪图 x_t = sqrt(ᾱ_t)·x_0 + sqrt(1-ᾱ_t)·ε
  5. 网络预测噪声 ε̂ = ε_θ(x_t, t)
  6. 损失 L = ||ε - ε̂||²
  7. 反向传播，更新网络参数
```

## 反向采样：从噪声到图片

训练完成后，采样算法：

```
x_T ~ N(0, I)  ← 从纯噪声出发

for t = T, T-1, ..., 1:
    z ~ N(0, I)  (t > 1 时才加；t=1 时 z=0)
    ε̂ = ε_θ(x_t, t)
    x_{t-1} = (1/sqrt(α_t)) · (x_t - β_t/sqrt(1-ᾱ_t) · ε̂) + sqrt(β_t) · z
```

每步只做一件小事：用网络预测的噪声 $\hat{\varepsilon}$，把 $x_t$ 轻微"去噪"成 $x_{t-1}$。1000 步后得到 $x_0$。

## 噪声调度（Noise Schedule）

$\beta_t$ 如何从 $t=1$ 增长到 $t=T$？

**线性调度（Ho 2020）**：$\beta_t$ 从 $\beta_1 = 0.0001$ 线性增长到 $\beta_T = 0.02$。
- 前期慢慢加噪（保留细节）
- 后期快速加噪（抹去粗糙结构）

**Cosine 调度（Nichol & Dhariwal, 2021）**：改进版，让 $\bar{\alpha}_t$ 按余弦曲线下降——前期加噪更慢，生成质量更好。

```python
# 线性调度
betas = np.linspace(0.0001, 0.02, T)

# Cosine 调度（改进版）
alphas_bar = np.cos((t/T + 0.008) / 1.008 * π/2) ** 2
```

## 为什么 DDPM 比 GAN 稳定？

| 问题 | GAN | DDPM |
|------|-----|------|
| 训练稳定性 | 两个网络博弈，容易崩溃 | 只有一个网络，监督学习 |
| 模式崩塌 | 严重 | 基本没有（每次采样随机路径不同）|
| 训练目标 | 对抗损失（难以优化）| MSE（简单直接）|
| 样本多样性 | 有限 | 高（噪声保证多样性）|
| 生成速度 | 快（一步）| 慢（需 1000 步）|

DDPM 的最大缺点：**采样慢**。1000 步意味着每张图需要运行神经网络 1000 次。

这个缺点直接催生了：
- **DDIM（2020）**：Song 等发现只需 50 步就能得到同等质量
- **Stable Diffusion（2022）**：在潜在空间（latent space）做扩散，进一步加速
- **DALL-E 2（2022）**：将 DDPM 与 CLIP 文本编码结合，实现文字→图片生成

## 数学推导：为什么可以直接跳到任意步 $x_t$？（选读）

这里用到**高斯分布的卷积性质**：两个高斯分布相加还是高斯分布。

设 $x_1 = \sqrt{\alpha_1} x_0 + \sqrt{1-\alpha_1} \varepsilon_1$，$x_2 = \sqrt{\alpha_2} x_1 + \sqrt{1-\alpha_2} \varepsilon_2$，代入：

$$
x_2 = \sqrt{\alpha_2\alpha_1} x_0 + \sqrt{\alpha_2(1-\alpha_1)} \varepsilon_1 + \sqrt{1-\alpha_2} \varepsilon_2
$$

后两项都是独立高斯，可以合并：方差为 $\alpha_2(1-\alpha_1) + (1-\alpha_2) = 1 - \alpha_2\alpha_1 = 1 - \bar{\alpha}_2$。

所以 $x_2 = \sqrt{\bar{\alpha}_2} x_0 + \sqrt{1-\bar{\alpha}_2} \varepsilon$。

用归纳法对任意 $t$ 都成立。这就是重参数化技巧能直接计算 $x_t$ 的数学基础。

## 历史地位与影响

DDPM 发表后的 3 年里，扩散模型几乎统治了图像生成领域：
- 2021：Improved DDPM（更好的调度）、Diffusion Beats GAN（超越 GAN 的 FID 分数）
- 2022：DALL-E 2、Stable Diffusion 1.x、Imagen（文生图）
- 2023：Stable Diffusion XL、一致性模型（更快采样）

与 GPT 系列并列，成为 2020 年代 AI 两大生成范式之一。
